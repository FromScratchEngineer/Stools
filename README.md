# ![alt text](assets/icons/stool-48.png) Stools
A firefox plugin that integrate Twitch chat in web based games for a more interactive experience.

## Current supported games
| Game | Status | Usage |
| --- | --- | --- |
| [Wordle](https://www.nytimes.com/games/wordle/index.html) | ✅ | Aggregate guesses from Twitch chat avoiding wrong guesses based on current game state. |
| [Scrandle](https://scrandle.com/) | ✅ | Display messages from Twitch chat over the choices based on simple word filter (left/right, 1/2)|

## Installation
1. [Download the latest release](https://github.com/fromscratchengineer/stools/releases/latest/download/stools.xpi) (`.xpi` file).
2. Open Firefox and go to the Add-ons page (`about:addons`, or `Menu > Add-ons and themes`).
3. Click on the gear icon and select "Install Add-on From File..."
4. Choose the downloaded file and install it.

## Setup
1. Go to the add-ons page (`about:addons`, or `Menu > Add-ons and themes`).
2. Find "Stools" in the list of installed add-ons and click on "..." and select "Preferences".
3. Enter your Twitch channel name and click "Save".

## Usage
1. Just open the supported game in your browser.
2. If you can't see anything, just click the Stools icon to toggle the overlay on/off.
